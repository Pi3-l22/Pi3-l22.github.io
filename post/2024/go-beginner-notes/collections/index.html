集合 Map

Map 是一种无序的键值对的集合。类似 python 中的字典。与切片 Slice 类似。

Map 最重要的一点是通过 key 来快速检索数据，key 类似于索引，指向数据的值。

Map 是一种集合，所以我们可以像迭代数组和切片那样迭代它。不过，Map 是无序的，遍历 Map 时返回的键值对的顺序是不确定的。

在获取 Map 的值时，如果键不存在，返回该类型的零值，例如 int 类型的零值是 0，string 类型的零值是 ""。

与切片一样，Map 是引用类型，如果将一个 Map 传递给一个函数或赋值给另一个变量，它们都指向同一个底层数据结构，因此对 Map 的修改会影响到所有引用它的变量。

定义 Map

可以使用内建函数 make() 或者 map 关键字来定义 Map。一般不太使用 make。

// 使用 make 函数
var map_variable =  make(map[KeyType]ValueType, initialCapacity)
map_variable := make(map[KeyType]ValueType, initialCapacity)
// 使用 map 关键字
var map_variable = map[KeyType]ValueType{}
map_variable := map[KeyType]ValueType{}

其中 KeyType 是键的类型，ValueType 是值的类型，initialCapacity 是可选的参数，用于指定 Map 的初始容量。

Map 的容量是指 Map 中可以保存的键值对的数量，当 Map 中的键值对数量达到容量时，Map 会自动扩容。如果不指定 initialCapacity，Go 语言会根据实际情况选择一个合适的值。

// 创建一个空的 Map
m := make(map[string]int)
m['apple'] = 1
m['banana'] = 2
m['orange'] = 3

// 创建一个初始容量为 10 的 Map
m := make(map[string]int, 10)
m['apple'] = 1
m['banana'] = 2
m['orange'] = 3

// 使用字面量创建 Map
m := map[string]int{
    "apple": 1,
    "banana": 2,
    "orange": 3,
}

注意：使用 map 关键字声明 Map 时，可以跟上 {} 直接进行初始化。若未在声明时使用 {} 初始化元素，与定义切片一样，程序不会分配内存，值为 nil，这时还需要配合 make() 进行初始化。也可以在后续通过获取和修改元素的方式来进行初始化。

var map_variable = map[KeyType]ValueType  // 创建 map
map_variable = make(map[KeyType]ValueType) // 初始化
// 等同于
// 创建并初始化 map
var map_variable =  make(map[KeyType]ValueType, initialCapacity)

map_variable := make(map[KeyType]ValueType, initialCapacity)

var map_variable = map[KeyType]ValueType{}

map_variable := map[KeyType]ValueType{}

获取元素：

// 获取键值对
v1 := m["apple"]
v2, ok := m["pear"]  // 如果键不存在，ok 的值为 false，v2 的值为该类型的零值

修改元素：

// 修改键值对
m["apple"] = 5

获取 Map 的长度：

// 获取 Map 的长度
len := len(m)

遍历 Map：

// 遍历 Map
for k, v := range m {
    fmt.Printf("key=%s, value=%d\n", k, v)
}

删除元素：

// 删除键值对
delete(m, "banana")

delete() 函数用于删除集合的元素, 参数为 map 和其对应的 key。

package main

import "fmt"

func main() {
    var siteMap map[string]string /*创建集合 */
    siteMap = make(map[string]string)

    /* map 插入 key - value 对,各个国家对应的首都 */
    siteMap [ "Google" ] = "谷歌"
    siteMap [ "Runoob" ] = "菜鸟教程"
    siteMap [ "Baidu" ] = "百度"
    siteMap [ "Wiki" ] = "维基百科"

    /*使用键输出地图值 */ 
    for site := range siteMap {
        fmt.Println(site, "首都是", siteMap [site])
    }

    /*查看元素在集合中是否存在 */
    name, ok := siteMap [ "Facebook" ] /*如果确定是真实的,则存在,否则不存在 */
    /*fmt.Println(capital) */
    /*fmt.Println(ok) */
    if (ok) {
        fmt.Println("Facebook 的 站点是", name)
    } else {
        fmt.Println("Facebook 站点不存在")
    }
}

// 输出结果
Wiki 首都是 维基百科
Google 首都是 谷歌
Runoob 首都是 菜鸟教程
Baidu 首都是 百度
Facebook 站点不存在
